# QA-agent Skills Inventory

这个目录是从当前机器的 OpenClaw 环境自动导出的技能清单，不只包含 QA skills，也包含非 QA 的系统/媒体/协作/知识库类 skills。

- Total skills: **65**
- Ready: **22**
- Needs setup: **43**
- QA skills: **11**
- Non-QA skills: **54**

## Files

- `ALL_SKILLS.md` — 人类可读的完整技能清单
- `all-skills-manifest.json` — 结构化 JSON 清单
- `qa-skill-suite-v1.zip` — 现有 QA 技能整包

## Notes

- `ready` 表示当前环境下已可直接使用。
- `needs setup` 表示还缺少 CLI、环境变量或配置。
- 这里列的是当前机器可见的全部 skill 元数据；并没有把 OpenClaw 自带 bundled skill 源码整包复制进仓库。
