# Permission and state prompts

## Permission prompts
- Can the user see it but not edit it?
- Can they edit after status changes?
- Can they export or download?
- Can they act on records created by others?
- Does API permission match UI visibility?

## State prompts
- Which actions are allowed in each status?
- Can a terminal state be reopened?
- What happens on repeated transition requests?
- What if state changes in another session while current user is acting?
