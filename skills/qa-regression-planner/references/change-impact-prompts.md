# Change impact prompts

## Direct impact
- Changed module itself
- Changed API endpoints
- Changed validation rules
- Changed permissions

## Indirect impact
- Upstream/downstream workflows
- Shared components and shared services
- Export/import, notification, callback, reporting
- Mobile/web/admin surfaces that reuse the same backend

## Historical prompts
- Which related bugs were fixed recently?
- Which modules are notoriously fragile?
- Which integrations fail when contracts drift?
