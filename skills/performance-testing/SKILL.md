# 性能测试技能

专业的性能测试工具，用于验证系统的响应速度、吞吐量和稳定性。

## 功能

- 响应时间测试
- 吞吐量测试
- 并发压力测试
- 负载测试
- 稳定性测试
- 资源使用率监控
- 性能瓶颈分析
- 性能基线建立

## 测试项目

### 1. 响应时间测试
- 平均响应时间
- 95%分位响应时间
- 99%分位响应时间
- 最大响应时间
- 最小响应时间

### 2. 吞吐量测试
- QPS（每秒查询数）
- TPS（每秒事务数）
- 带宽测试
- 数据处理能力

### 3. 并发测试
- 并发用户数
- 并发连接数
- 并发请求数
- 并发峰值测试

### 4. 负载测试
- 正常负载
- 峰值负载
- 超载测试
- 恢复测试

### 5. 稳定性测试
- 长时间运行
- 内存泄漏
- 资源耗尽
- 异常恢复

## 监控指标

```
- CPU使用率
- 内存使用率
- 磁盘I/O
- 网络带宽
- 连接数
- 错误率
- 超时率
```

## 输出格式

```markdown
### 性能测试结果

| 指标 | 目标 | 实际 | 状态 |
|------|------|------|------|
| 平均响应时间 | <200ms | 150ms | ✅ 通过 |
| 95%响应时间 | <500ms | 420ms | ✅ 通过 |
| QPS | >1000 | 1250 | ✅ 通过 |
| 并发数 | >100 | 150 | ✅ 通过 |
| 错误率 | <1% | 0.2% | ✅ 通过 |

### 性能分析
- 瓶颈1：xxx
- 优化建议：xxx
```
