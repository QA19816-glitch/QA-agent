# Release gate rubric

## GO
- Core path green
- No unresolved showstoppers
- Residual risk acceptable for release plan

## GO WITH CONDITIONS
- Limited known issues with accepted workaround
- Rollout guardrails or monitoring required
- Scope-limited release or feature flag is available

## NO-GO
- Core path broken
- Severe unresolved defect remains
- High confidence is impossible due to missing coverage or unstable environment
