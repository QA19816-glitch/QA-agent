# best-prompts

本 skill 基于 GitHub 仓库 `xstongxue/best-prompts` 整理，用于在常见工作场景下快速定位和复用高质量 Prompt。

## 适用场景

- 学位论文与小论文写作
- 需求分析、方案设计、代码实现、代码审查、Bug 修复
- PPT / Draw.io / 图表生成
- 自媒体写作、公众号文章、封面图、插图
- 项目文档整理、工作汇报、简历优化

## 使用建议

1. 先判断用户要解决的具体场景（论文 / 开发 / PPT / 自媒体 / 文档）。
2. 从 `references/prompts/` 下选择最贴近的 prompt 模板。
3. 必要时按用户上下文做轻量改写，而不是完全重写。
4. 若用户希望沉淀成自动调用能力，可进一步将其中稳定模板改造成专门 skill。

## 目录说明

- `references/README.md`：原项目总说明
- `references/prompts/`：各类高质量 prompt 模板

## 典型用途

- “给我一个高质量论文写作 prompt”
- “帮我找一个需求分析 prompt”
- “帮我找做答辩 PPT 的 prompt”
- “我要写公众号文章，有没有现成 prompt”
- “把这个项目经历整理成简历描述 prompt”
