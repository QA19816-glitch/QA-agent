# Common web flow patterns

## Account flows
- login
- logout
- password reset
- session timeout / re-login

## CRUD flows
- create
- save draft
- edit
- delete / archive
- list and detail consistency

## Commerce / transaction flows
- search
- add to cart / selection
- submit
- payment / confirmation
- cancellation / refund if applicable

## Workflow flows
- submit for approval
- approve / reject
- resubmit
- audit trail visibility
