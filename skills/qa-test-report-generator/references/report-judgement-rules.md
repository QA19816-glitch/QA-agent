# Report judgement rules

## Recommend release
- Core flows passed
- No open S0/S1 defects
- Residual risks are acceptable and documented

## Conditional release
- Non-core defects remain
- Workarounds exist
- Scope limits or env blockers are known and accepted

## Do not recommend release
- Core path failure remains
- Data integrity/security risk remains
- Coverage too incomplete to support release confidence
