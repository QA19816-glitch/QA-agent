# SEO测试技能

专业的SEO测试工具，用于验证网站的搜索引擎优化配置。

## 功能

- 验证网页Title、Description、Keywords设置
- 检查SEO参数完整性
- 验证TDK规则配置
- 检测SEO友好性
- 验证noindex/index设置
- 检查Robots.txt配置

## 使用场景

- 测试网页SEO参数是否正确
- 验证SEO配置是否符合规范
- 检查TDK长度是否在限制内
- 验证搜索引擎收录设置
- 批量检查多个页面的SEO

## 测试项目

### 1. TDK基础检查
- Title长度检查（50字符内）
- Description长度检查（150字符内）
- Keywords长度检查（100字符内）
- 必填字段完整性检查

### 2. SEO规则验证
- 默认TDK规则验证
- 公共配置验证
- 自定义规则验证
- 一键生成规则检查

### 3. 收录设置检查
- index/noindex设置
- Robots.txt配置
- 敏感页面保护
- 搜索结果页设置

### 4. SEO友好性检查
- 标题是否包含关键词
- 描述是否完整
- 关键词是否合理
- 页面是否可被收录
