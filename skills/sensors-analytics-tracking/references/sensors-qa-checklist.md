# 神策埋点测试清单

## 1. 事件是否触发
- 核心页面曝光是否上报
- 关键点击是否上报
- 成功/失败结果是否都上报

## 2. 触发时机是否正确
- 页面进入即触发还是首屏渲染后触发
- 点击前还是点击后触发
- 成功回调后再发还是请求发出即发

## 3. 属性是否完整
- 必填属性是否齐全
- 可选属性是否有脏值
- 枚举值是否符合约定

## 4. 属性值是否正确
- business_id / order_id / user_id 是否串值
- 页面名称、渠道、来源是否正确
- 登录态和匿名态是否正确区分

## 5. 是否有重复 / 漏报
- 连点是否重复上报
- 页面反复渲染是否重复曝光
- 返回/刷新/重试是否丢失或重复

## 6. 身份体系是否正确
- 未登录 distinct_id
- 登录后 identify / login 行为
- 退出登录后身份切换
- 多账号切换是否串用户

## 7. 平台差异
- H5 与 App 内嵌 H5 是否重复上报
- Android / iOS 是否一致
- 弱网 / 离线重发是否符合预期
