# QA-agent 技能商店

> 一个以 **测试 / QA 技能** 为核心的 OpenClaw 技能仓库。\
> 你可以把它直接分享给同事：别人进来后可以按分类看技能，或者直接下载压缩安装包，一键安装到自己的 OpenClaw。

## 🎯 你可以直接从这里开始

### 1）按分类找技能
- **测试专项技能**：[`INSTALL_GUIDE/TESTING_SPECIALTIES.md`](./INSTALL_GUIDE/TESTING_SPECIALTIES.md)
- **非测试技能**：[`INSTALL_GUIDE/NON_TESTING_SPECIALTIES.md`](./INSTALL_GUIDE/NON_TESTING_SPECIALTIES.md)
- **全部技能目录**：[`INSTALL_GUIDE/ALL_SKILLS.md`](./INSTALL_GUIDE/ALL_SKILLS.md)

### 2）不知道怎么选？直接看推荐安装包
- **建议安装包总览**：[`INSTALL_GUIDE/RECOMMENDED_INSTALL_PACKS.md`](./INSTALL_GUIDE/RECOMMENDED_INSTALL_PACKS.md)
- **技能商店首页**：[`INSTALL_GUIDE/README.md`](./INSTALL_GUIDE/README.md)

### 3）直接下载压缩包
- **压缩安装包下载页**：[`dist/install-packs/README.md`](./dist/install-packs/README.md)

## 🧭 仓库结构
- `INSTALL_GUIDE/`：中文目录页 / 技能商店 / 一键安装脚本
- `skills/`：所有 skills 源码

- `inventory/`：历史与机器清单（已精简）
