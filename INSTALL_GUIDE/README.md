# QA-agent 技能商店

把这个仓库当成一个 **技能商店** 来用就行：只看 4 个主入口，不需要在一堆目录里翻。

## 四个主入口

### 1）总包
适合：想一次拿全量 skills 的人。
- 文档：[`ALL_SKILLS.md`](./ALL_SKILLS.md)
- 下载：<https://raw.githubusercontent.com/QA19816-glitch/QA-agent/main/dist/all-skills-bundle-v1.zip>

### 2）测试包（QA）
适合：所有做测试 / QA / 自动化 / 性能 / 回归 / 缺陷的人。
- 文档：[`TESTING_SPECIALTIES.md`](./TESTING_SPECIALTIES.md)
- 下载：<https://raw.githubusercontent.com/QA19816-glitch/QA-agent/main/dist/testing-specialties-skills-bundle-v1.zip>

### 3）非测试包
适合：文档、协作、Agent 工具、办公效率、飞书、通用工具等非测试场景。
- 文档：[`NON_TESTING_SPECIALTIES.md`](./NON_TESTING_SPECIALTIES.md)
- 下载：<https://raw.githubusercontent.com/QA19816-glitch/QA-agent/main/dist/non-testing-specialties-skills-bundle-v1.zip>

### 4）建议安装包
适合：大多数人先装一套通用核心能力。
- 文档：[`RECOMMENDED_INSTALL_PACKS.md`](./RECOMMENDED_INSTALL_PACKS.md)
